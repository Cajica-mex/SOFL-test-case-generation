# SOFL-test-case-generation
Generación automática de casos de prueba a partir de especificaciones formales escritas en el lenguaje SOFL
El archivo "SOFL-test-case-generation" es el archivo principal, en donde se mandan a llamar las especificaciones en SOFL.
El archivo "PSO_library" es un modulo en donde se implemento el algoritmo PSO, este modulo se manda a llamar desde el archivo principal "SOFL-test-case-generation"
